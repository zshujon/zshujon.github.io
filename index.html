Research Outcome: Datasets

  1. 40k Bangla Newspaper Article

    Custom-made Bangla text corpus from various online newspapers. The Dataset may be used for Text Generation, Document Classification, Topic Modeling, etc.

    https://www.kaggle.com/zshujon/40k-bangla-newspaper-article

  2. 4K Bangla Song Lyrics

    The dataset contains 4K (4068) Bangla song lyrics in Bangla Unicode font. All the lyrics were collected via web scraping. Those who want to use this must do some cleaning. The dataset is useful for character-by-character sequence generation or any other NLP task.
